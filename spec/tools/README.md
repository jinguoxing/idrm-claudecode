# AI工具配置

本目录包含三个AI工具的配置模板和使用指南。

## 工具清单

- **Cursor** - IDE集成，实时补全
- **Claude CLI** - 命令行，批处理
- **Kiro.dev** - 结构化开发，团队协作

## 配置文件

### Cursor
- `.cursorrules.template` - 配置模板
- 使用指南：`../../Cursor使用指南.md`

### Claude CLI  
- `.clinerules.template` - 配置模板
- 使用指南：`../../ClaudeCodeCLI使用指南.md`

### Kiro.dev
- `steering.template.md` - Steering配置模板
- 使用指南：`../../Kirodev使用指南.md`

## 如何使用

1. 复制对应的template到项目根目录
2. 根据实际情况调整
3. 参考使用指南开始使用
